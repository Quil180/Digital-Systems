\subsection{Convert $(58.27)_{10}$ to Hexadecimal then Binary}
    \begin{center}
    $58/16 = 3 R 10$\\
    $3/16 = 0  R  3$\\
    \\
    $0.27 * 16 = 4.32$\\
    $0.32 * 16 = 5.12$\\
    $0.12 * 16 = 1.92$\\
    $0.92 * 16 = 14.72$\\
